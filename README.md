# Zenith Dental — Premium Dental Clinic Website Template

Zenith Dental is a state-of-the-art, premium website template designed specifically for dental clinics, dental surgeons, orthodontists, and healthcare practitioners. Built with a modern aesthetic, standard HTML5 structure, responsive layout controls, and an advanced client-side booking wizard, Zenith Dental provides everything needed to establish a premium digital medical presence.

---

## 🌟 Key Features

* **Live Interactive Booking Wizard**: A highly polished, multi-step appointment scheduler allowing users to choose clinical treatments, dentists, pick calendar slots, fill in patient information, and view an instant consultation summary.
* **Modern Design & Rich Aesthetics**: Harmonious, clinical-grade color palette (Clinical Teal & Sky Blue accents), premium Google typography (Outfit and Inter), smooth micro-animations, glassmorphic header, and scroll-reveal triggers.
* **100% Mobile & Responsive Grid**: Designed and thoroughly tested across all devices (320px to 1920px viewports) including desktop monitors, tablets, and smartphones.
* **WCAG AA Accessibility Compliance**: Integrated focus trapping on modals, keyboard-friendly outline states, full ARIA controls for screen readers, and optimized color contrast ratios.
* **SEO Optimized Structure**: Includes JSON-LD local business schema, canonical routing placeholders, Twitter Card and Open Graph sharing metadata, along with semantic HTML outline checks.
* **Performant WebP Assets**: Fully compressed and converted media files reducing site load times and matching optimal speed scores.
* **Interactive Services details**: Modal window logic programmatically driven from centralized data records.

## 🛠️ Customization & Setup

### 1. Centralized Data Customization
You can easily change services, clinicians, client reviews, and FAQ items without editing complex HTML. Open `Template/assets/js/data.js` and modify the fields inside the objects.

### 2. Clinic Details and Text Markup
Search for `EDITABLE` in `Template/index.html` to find direct elements containing telephone lines, geographical addresses, clinic operational hours, and social media handles.

## 📋 Technology Stack

* **Structure**: Semantic HTML5 Markup
* **Styles**: Vanilla CSS3 Custom Variables (Custom properties-based design system)
* **Scripts**: Modern Vanilla JavaScript (ES6 Modules/defer-loaded scripts)
* **Fonts**: Outfit (Headings) and Inter (Body copy) via Google Fonts
* **Icons**: Inline scalable SVGs (No external icon packages needed for optimal performance)
